    Returns the formatted and localised date, time or datetime for a given Date, Time, DateTime or struct with the appropriate fields.

    Arguments

    • date is a Date, Time, DateTime or other struct that contains the required date and time fields.

    • format is a valid format string, for example yy/MM/dd hh:MM

    • locale is any valid locale name returned by Cldr.known_locale_names/0 or a Cldr.LanguageTag struct. The default is Cldr.get_locale/0

    • options is a keyword list of options. The valid options are:

    Options

    • :number_system. The resulting formatted and localised date/time string will be transliterated into this number system. Number system is anything returned from inspect(unquote(backend)).Number.System.number_systems_for/1

    NOTE This function is called by Cldr.Date/to_string/2, Cldr.Time.to_string/2 and Cldr.DateTime.to_string/2 which is the preferred API.

    Examples 

    iex> Elixir.CoreWeb.Cldr.DateTime.Formatter.format %{year: 2017, month: 9, day: 3, hour: 10, minute: 23},
...> "yy/MM/dd hh:MM", "en"
{:ok, "17/09/03 10:09"}
