    Core.DataDonation.Tools (core v0.1.0) View Source

    A data donation allows a researcher to ask participants to submit data. This data is submitted in the form of a file that is stored on the participants device.

    Tools are provided that allow for execution of filtering code on the device of the participant. This ensures that only the data that is needed for the study is shared with the researcher.
