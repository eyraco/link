    Specs

    date_time_available_formats(
  Cldr.Locale.locale_name() | Cldr.LanguageTag.t(),
  calendar()
) :: {:ok, formats()}

    Returns a map of the available non-standard datetime formats for a given locale and calendar.

    Arguments

    • locale is any locale returned by Cldr.known_locale_names/0 or a Cldr.LanguageTag.t()

    • calendar is any calendar returned by Cldr.DateTime.Format.calendars_for/1 The default is :gregorian
