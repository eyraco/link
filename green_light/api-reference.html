    API Reference green_light v0.1.0

    Modules

    GreenLight

    Defines an authorization module.

    GreenLight.Access

    Helper functions to check if a given user can access / invoke something.

    GreenLight.AuthorizationNode
    GreenLight.Config
    GreenLight.Ecto.Query

    Query functions around role assignments.

    GreenLight.Ecto.RoleAssignment

    A role assignment entity is used to assign a role to a principal on a specific entity.

    GreenLight.Live

    The Live module enables automatic authorization checks for LiveViews.

    GreenLight.Loaders

    This is module that automates the loading of entities based on path parameters.

    GreenLight.PermissionMap

    The permission map holds a mapping of permissions to roles. The permission map is intended to be an implementation detail of the authorization framework. Convenience functions to build it are provided in other modules.

    GreenLight.Permissions

    The permissions module provides several functions that create permission strings and GreenLight.PermissionMaps.

    GreenLight.Plug

    This Plug validates the permissions before the request is allowed to proceed.

    GreenLight.Principal

    The principal is used as a representation for any person / thing that can interact with the system.
